never add "co-authored-by: claude" or any claude reference in commit messages or code comments.

**why:** user wants commits to look fully human-authored. any mention of claude in commits is unwanted.

**how to apply:** strip the co-authored-by line from every commit message. keep messages short and human-sounding.
